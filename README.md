# SimpleClock

A simple Clock web application (AI generated)

## 📱 Live Demo

Check out the live version here: **[SimpleClock GitHub Pages](https://m3koenig.github.io/SimpleClock/)**

## 📋 Description

SimpleClock is a lightweight, user-friendly web application that displays the current time. Built with HTML, CSS, and JavaScript, it provides a clean and minimalist interface for viewing the time.

## ✨ Features

- **Real-time Display**: Shows the current time updated in real-time
- **Simple Interface**: Clean and minimal design
- **Responsive Design**: Works on different screen sizes
- **AI Generated**: Crafted with AI assistance for quick development

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/m3koenig/SimpleClock.git
```

2. Navigate to the project directory:
```bash
cd SimpleClock
```

3. Open `index.html` in your web browser or visit the [GitHub Pages](https://m3koenig.github.io/SimpleClock/) link.

## 🎯 Usage

Simply open the application in your browser to see the current time displayed. The time updates automatically as seconds pass.
